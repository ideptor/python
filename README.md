# python

가상환경 설치
```
$python -m venv 가상환경이름
```


가상환경 실행
```
$./가상환경폴더/Script/activate
```
보안에러 발생시 PowerShell 아이콘 우클릭 → 관리자모드로 실행 후 (관리자 모드로 실행하면 System32에서 시작)



실행화면
```
(가상환경이름)$pip list
```

가상환경 종료
```
(가상환경이름)$deactivate
```
